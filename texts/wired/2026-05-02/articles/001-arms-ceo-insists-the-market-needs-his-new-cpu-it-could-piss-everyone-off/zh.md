# Arm 首席执行官坚持认为市场离不开他的新 CPU，这可能会得罪所有人

Section: Lauren Goode
Date: Mar 25, 2026 6:00 AM

Rene Haas正半躺在他位于加州圣何塞办公室的沙发上，手里拿着一个篮球，几乎挡住了他的脸。WIRED 的摄影师第一次让他摆出这个姿势时，Haas就皱起了眉头。“我马上就想到标题了：‘大家会说 Arm 的 CEO 在工作时睡觉了’，他笑着说。

但Haas还是配合了。他给了我们46分钟时间，随后把我们请了出去，让他可以接入一个与软银首席执行官、也是 Arm 董事会主席的孙正义的电话。

我是在芯片公司发布重大消息的前几天见到Haas的，这家公司将推出自有的硅片。对于一家主要靠把自己的架构授权给其他芯片公司、从未自己制造芯片的企业来说，这一步是一次巨大的赌注。Apple、Tesla、Nvidia、Microsoft、Amazon、Samsung 和 Qualcomm 都基于 Arm 的架构生产或销售芯片，要么直接许可芯片设计，要么向该公司支付版权费。人们估计，世界上每个人平均拥有大约三颗 Arm 芯片。

从另一个角度看，造芯片其实是 Arm 回归其根基。公司可以追溯到20世纪70年代后期，当时有两位计算机架构师创建了 Acorn Computers（松果电脑公司），推出了一款基于 RISC 架构的微处理器。到了1990年代初，这家公司陷入困境，时任 CEO 转向将其设计授权给其他公司。快进到2010年代中期，Arm 的高能效移动芯片设计帮助其成为世界上最重要的芯片知识产权公司。

Arm的发展轨迹并非一帆风顺。2016年，软银收购Arm并将这家上市公司私有化后，智能手机市场增速放缓。Arm不得不大幅推进新的业务线。2020年，英伟达（Nvidia）曾试图收购它，但交易被监管机构叫停。随着这笔交易在2022年宣告破裂，哈斯（Haas）接过首席执行官一职。他让Arm再次上市，而软银仍持有公司90%的股份。

哈斯于2013年从英伟达加入Arm，曾负责该公司的计算产品事业部，后来接管了Arm的“摇钱树”——IP产品部门。和英伟达首席执行官黄仁勋依靠其几十年的行业阅历来开讲似乎很自然——比如围着火堆对着孩子们说起并行计算的早期故事——一样，哈斯在被问及当前事件是否让他担心业务时，也会迅速提及20世纪80年代的地缘政治动荡。答案是：不会担心。他告诉我，自己已与唐纳德·特朗普总统见过面近六次，但他并不特别担心美国政府干预其英国公司的事务。他身材高挑，却不算威严，通常穿着带有小跟的圣罗兰（Saint Laurent）靴子、夹克和一块帕内莱（Panerai）手表。

芯片行业内幕人士表示，63岁的哈斯是个社交手腕非常娴熟的人，和硅谷最顶尖的科技大佬都能打成一片。华尔街日报曾称他是“天生的外交官”。但在这项芯片项目上，Arm——以及哈斯本人——却冒着得罪公司最忠诚合作伙伴的风险，而这也是硅谷所谓“最不保密”的事情之一。你能与人长期在礼貌的晚宴上维持友好关系，之后却宣布“我要买下你们的房子”吗？哈斯似乎确信自己可以做到。

本次访谈经过压缩并做了轻度编辑，以提高清晰度。

劳伦·古德：你接任CEO之后，人们都说公司文化发生了很大变化。你认同这种判断吗？

雷内·哈斯：我学到的一件事——我在为詹森工作时就本能地明白了这点，但我在接手这里后更是把它内化为信条——就是CEO会为公司定下基调。

我的成长经历，最终塑造了我作为领导者的样子，确实是在三十年前搬到硅谷、先后在几家创业公司任职，后来在英伟达工作的过程中被快速加速的。所有这些公司的共同点，是我都在为创始人打过工。当时我说不出“哦，在创始人身边工作正是我向往的环境”这样的话；但回想起来，我认为正是在那里塑造了我的职业 DNA，也是在那儿我找到了自己最能 thrive 的环境。

那到底是什么样的环境呢？是敢于冒险、勇于扩张、市场节奏快、允许犯错，同时又愿意下注于重大战略方向。正因为如此，当我在这里接管公司时，我就明确说：这就是我想要的文化。

你上任首席执行官时，这个 Arm 芯片项目已经在进行中了吗？

其中有多少是你的想法，有多少是 Masa 的想法？

这是他做的决定吗？不是。我是 CEO。 他是否对我们讨论过的各种权衡和事项非常清楚并且参与其中？百分之百。 但这更像是一次头脑风暴式的合作，像想法上的搭档，而不是你的老板对你说“这个要，那个不要”。他不太会介入到那种细节层面。

SoftBank 持有 ARM 90%股份带来的一个好处是他是我们的董事长，也是最大股东，因此他是我几乎每天都在联系的人。我们关系非常密切。

一位分析师告诉我，你每天会和 Masa 通话 10 到 12 次。

有些日子会这样。他确实喜欢打电话。我是全天候在线。他很清楚我的作息安排。他知道我几点起床，知道我的锻炼时间表。他知道我几点睡觉。这些他都很熟悉。

是什么让你认为现在就是 Arm 推出这颗芯片的时机？

所以在这个过程中，我们从一家 IP 公司转变成了一家计算平台公司。我的意思是，当你思考 CPU 在任何生态系统中的作用时，你会发现 CPU 硬件与软件生态之间存在如此强的相互依赖，无论是在 Windows、macOS、iOS、Android 还是 Linux 上运行。我想不需要强调的是，我们并未把这点讲清楚，也许我们以前没有意识到它的重要性；但当我接手时，我非常明确知道这就是我们的身份，也正是我们必须推进的方向。

那么为什么我们要造一颗芯片？当你是一家计算平台公司时，有些时候，生态系统确实需要你亲自打造一个实物产品。过去我们见过这种情况：无论是微软打造 Surface 笔记本以支持 Windows 生态，还是说谷歌造 Pixel 手机，但同时三星仍在生产 Android 手机，惠普、戴尔、联想也都在造笔记本。

那些产品仍然只占他们业务的一小部分。

但这确实有利于生态系统。如果你考虑 Gemini 在 Pixel 平台上的情况：若 Gemini 在 Android 上被优化，就会帮助三星。再看微软为 Surface 开发工具或应用，这会让 Windows 受益，也会让惠普、联想和戴尔受益。

这正是我认为大家并没有总是理解我们的地方。他们说："哦，它只是一家知识产权公司。"但我们交付的知识产权是一种计算平台，而归根结底，平台公司需要做一些事情来推动生态系统向前发展。

这块新芯片叫什么？

它就是 Arm AGI CPU。

AGI 是指“人工通用智能”吗？

没错。你喜欢吗？

我并不反感它。我在想AGI这个词是否会在某个时候退出时尚，因为行业会围绕某些其他超智能的概念重新归并。

我打赌它不会。

这个产品的目标客户是谁？

第一批客户将是Meta。但我们也有SK Hynix、Cisco、SAP和Cloudflare。我们有多个客户。

而这是一款为数据中心而设计的芯片。

是什么让这款芯片与众不同？

有好几点原因，其中之一是它的能效惊人。该公司最初是从为电池供电或手机里运行的芯片开始起家的，所以它有着对效率的执着思维。你听到关于人工智能增长时，都会听到一个共同点：这需要消耗多少能源。因此，只要能做出世界上最节能的服务器CPU，这就是一件大好事。这将是最具能效的CPU。

另一件它会表现出色的事情是运行“代理式AI”。外界长期流传的一个误解是，随着AI的兴起，GPU或加速器才是全部。但当你看看智能体在数据中心里的实际运行方式时，那是只有CPU才能完成的工作。GPU并不会消失，但要支撑这些智能体的运行，反而需要更多的CPU。

Meta 刚刚与 Nvidia 和 AMD 签下了两笔相当大的交易，涵盖了 CPU 和 GPU 两类芯片。这其中包括 Nvidia 的新一代 Vera CPU，它属于更大的 Vera Rubin 超级芯片方案。

没错，也就是 Arm。

请原谅我直白一点，推出你自家的 CPU 会不会惹恼像 Nvidia 这样的客户？

是啊，所以回到平台讨论……这会让所有人的船都抬高。

这是一个非常方便的答案。

但这并不算不准确。写给 Arm 的软件越多，就会有越多针对 Arm 的优化——这一切都能帮助所有基于 Arm 建设的人。这个机会是巨大的。放到数据中心和 AI 增长相关的资金规模上看，规模实在是非常大。至于你问它会不会惹恼英伟达？我想它更可能会惹恼英特尔和 AMD，而不是英伟达。

因为这会从他们那里抢走市场份额。

你指的是英特尔的 x86 架构和 Arm 架构之间的差别。也就是说，你不认为这会惹你那位朋友 Jensen 生气，但 AMD 和英特尔可能会对此做出一些回应。

我把“piss off”当作一种带点俏皮的说法。为 Arm 生态建设芯片对整个生态有利，也对 Jensen 有利。如果你有 Nvidia 的 Vera 芯片，这是个很棒的产品；而你还拥有 Arm AGI CPU，这也是个很棒的产品，那么对 Intel 和 AMD 来说就没那么美好——我能说的只有这些。

Nvidia 到底会买吗？你们活动里有 Amazon 的高管在视频中为你们的芯片背书——会买单吗？毕竟 Amazon 早已有自己的芯片。

如果 Jensen 买些当然再好不过，但 Vera CPU 已经与他自己的芯片紧密耦合，他们也有自己的联网技术来连接所有芯片。而 Amazon 大概会继续使用他们自己的 Graviton 芯片。

但我们不是为 Amazon 开发这个产品，我们开发它是因为我们认为 Arm CPU 的市场真的非常大。它本质上是一个长期供需不足的市场。如果你看看我们的一些早期客户，比如 Meta，他们会在风冷机架中使用我们的芯片——他们不需要 Nvidia 的 NV Link。再比如 Cloudflare——他们买不起 Amazon 的 Graviton 芯片，而且也没有能力自己制造。

谁在为你制造芯片？是联合封装吗？是网络吗？

台积电。全部都来自台积电。我们也在打造一套服务器参考设计。不是“我们自己”，而是与超级微电脑（Super Micro）和富士康（Foxconn）等合作伙伴一起合作。放在今天这个时代，单独做一颗芯片是不够的。所以我们与生态系统里的合作伙伴共同努力，以服务器机架的形式交付一个完整方案。

你们现在将不得不回答产量、良率以及毛利率方面的问题。

是的。2013年我从英伟达加入Arm时，我记得当时是这样想的："哇，这里没有报废，没有良率问题，没有退货处理（RMA），也没有预测。你只要点击发送，版税报告就会出来。多么美妙的生意啊。"

所以我心里有一部分会想：‘我是不是又要重操旧业了？’没错，这些都将是我们现在需要处理的事务，但我知道如何去做。我知道需要做什么，而且我们已经在公司内部从运营和上市推进的角度组建了一支团队，建立了支持这种流程所需的一切。

你们为这项工作投入了多少员工？

我们新增了大约2,000名工程师，负责后端设计、实现和子系统工作。他们并不都在做这颗芯片，但我们把许多已经在计算子系统上完成的工作都用于了这类任务。我不确定要不要报出一个确切数字，说这是一个500人项目还是700人项目，因为它依赖于我们已经完成的许多工作。

业内很多人认为，一款芯片刚刚推出时，通常需要两到三个代际，才能产生你所期望的影响。你同意这种看法吗？

是的。自从我担任 CEO 以来，我们一直在构建我们称之为“计算子系统”的东西。它本质上是把计算引擎的全部基础模块先搭好，再交给客户，让他们据此构建自己的片上系统（SoC）。在某些场景下，里边大约有 85% 的工作已经替他们完成了。这是非常庞大的工程量。

Arm AGI CPU 所基于的这套计算子系统，已经由其他合作伙伴在硅片上量产并交付过了。所以我们对这颗芯片能否正常运作的把握非常高。你说得对，它可能需要经过一定次数的迭代。如果你要进入一个全新的市场，并且尝试做一件前所未有的产品，确实会多一些迭代。它以前从未经过验证，你也很难准确预判市场会如何接受。

而这一次，我们对“第一颗样品”是否会非常好、是否能成为量产候选芯片的信心，确实高到非常高。

上帝啊，赐我一位芯片 CEO 的信心吧。好，我来问你一个快问快答。我会对你说几个词，你只要给我一个快速反应。英特尔。

还算是起步阶段？它诞生于2014年。

我只给你一个词：起步阶段。

我不知道你是否碰巧读过我同事去年写的《WIRED》文章，内容是关于RISC的历史以及RISC-V的成立。

而且该社区里有些人——我不想直接用那个词——对Arm可不怎么友好。

他们说你们这些人是混蛋。

是他呢，还是我？（他朝自己的参谋长萨米尔·沙阿[Saumil Shah]指了指。）我们也可以这样说他们。

回到快问快答：Sam Altman。

你凭什么认为他很聪明？

我认识他。他是那种做事会着眼于长期布局的人。的确，他想得非常长远。谈到他如何看待问题、如何看待技术，他都显得很有远见。是的，我认为他很聪明。

当你看政治、地缘政治、宏观经济环境，以及内存短缺——在这样的时刻推出一颗芯片，简直太疯狂了。等你准备把这款芯片推向市场时，这些因素有没有改变你的计划？

没有改变。因为对我们来说，这是一段漫长的征程。Arm 的 AGI CPU 是我们的第一款产品。现在仅仅是个开始。

我大学毕业后的第一份工作是在1984年于德州仪器（Texas Instruments）。那一年是半导体行业的繁荣年。可是，20世纪80年代初却是有史以来最严重的衰退之一。然后在1985年，里根上台连任后，削减了国防开支，而这曾是TI业务的一大支柱。

我从中学到的教训是什么？就是任何事情都没有“最佳时机”。认真说，做任何事都没有一个完美时机；同样也没有什么事情是“永远不该做”的。你只需要去做。

我听说你很崇拜著名篮球教练菲尔·杰克逊。为什么？

我喜欢菲尔·杰克逊的地方在于，他接手了两支很有天赋、实力很强的球队，却把它们打造成了冠军队伍。迈克尔·乔丹在这之前并不是冠军——

迈克尔·乔丹在北卡罗来纳州大学时只是一个脱颖而出的球员。

他在北卡罗来纳州赢得了这个头衔，但他并不是球队里最优秀的球员。科比·布莱恩特在菲尔·杰克逊之前也不是一名冠军球员。他把他们变成了冠军。

你见过菲尔·杰克逊吗？

这怎么还没发生呢？

【朝自己的参谋长点头】有人不会做自己的本职工作。

作为一名领导者，最让你烦恼的是什么？请不要说“愚蠢的问题”。

那些对现状感到满意的人。

那有什么例子吗？

“我们一直是这样做的。不能确定自己能做得更好。看不出还有别的改进办法。”

风险的反面。作为一名领导者，是什么激励着你？

做出巨大的影响。去做能够改变世界的事情。

在硅谷，大家都这么说。这到底是什么意思？

是啊，这到底是什么意思？嗯，如果我把它归结到Arm的话，激励我的，是我们正在用这款芯片做的事。这是对公司来说一次非常、非常重大的变化。

这是“押上公司、孤注一掷”式的一刻吗？

那也就是说，即使这颗芯片失败了，你仍然对自己的知识产权业务充满信心吗？

当然。并且这颗芯片会成功。它一定会成功。
